# Template Wireframe - Guide de création

**Date** : 2025-12-23

---

## Structure d'un wireframe

### Éléments à inclure

1. **Header/Navigation**
   - Logo
   - Menu de navigation
   - CTA principal

2. **Hero Section**
   - Titre principal (H1)
   - Sous-titre/Description
   - CTA principal
   - Image/Visuel (placeholder)

3. **Sections de contenu**
   - Titre de section (H2)
   - Contenu (texte, cartes, listes)
   - CTAs secondaires

4. **Footer**
   - Liens navigation
   - Contact
   - Réseaux sociaux
   - Legal

### Annotations à ajouter

- **Dimensions** : Largeur, hauteur des sections
- **Espacements** : Marges, paddings
- **Interactions** : Hover, click, scroll
- **Responsive** : Notes sur l'adaptation mobile/tablette
- **Contenu** : Placeholders pour textes et images

---

## Breakpoints à considérer

### Mobile
- **375px** : iPhone standard
- **414px** : iPhone Pro Max

### Tablette
- **768px** : Tablette portrait
- **1024px** : Tablette paysage

### Desktop
- **1440px** : Desktop standard
- **1920px** : Desktop large

---

## Principes à respecter

1. **Mobile-first** : Commencer par mobile
2. **Hiérarchie claire** : 1 action principale par section
3. **Parcours de conversion** : CTAs visibles et accessibles
4. **Accessibilité** : Navigation clavier, contraste
5. **Performance** : Contenu prioritaire visible sans scroll

---

## Checklist avant validation

- [ ] Tous les éléments nécessaires présents
- [ ] Annotations complètes
- [ ] Responsive considéré (mobile, tablette, desktop)
- [ ] Parcours de conversion clair
- [ ] Accessibilité prise en compte
- [ ] Cohérence avec l'architecture de l'information Phase 1
